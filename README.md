# PayWallet

PayWallet is a full-stack digital wallet and peer-to-peer payment application that enables users to securely manage funds, transfer money, and track financial activity through an interactive dashboard.

The application provides user authentication, wallet management, money transfers, transaction tracking, profile management, and financial analytics in a responsive and user-friendly interface.

## Key Features

* Secure user registration and login using JWT Authentication
* Digital wallet with balance management
* Add money to wallet (Top-up functionality)
* Send money to other registered users
* Transaction history with search and filtering
* Recent contacts for quick transfers
* Profile management with profile picture upload
* Password change functionality
* Analytics dashboard with charts and transaction insights
* Monthly spending and activity tracking
* Export transaction history to CSV
* Responsive design for desktop, tablet, and mobile devices
